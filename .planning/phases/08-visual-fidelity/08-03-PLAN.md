  <action>
Replace the existing seat-related custom properties with cloud_2-aligned values, and add two new tokens consumed by Plans 08-01 and 08-02.

In `src/styles/tokens.css`, locate the `:root` block containing seat tokens and update as follows:

```css
/* Seat color palette — aligned to cloud_2 reference (verified 2026-04-27) */
--color-seat-available:        #c8e6c9;  /* cloud_2 plane_m_s_a — pastel green */
--color-seat-occupied:         #ef5350;  /* cloud_2 plane_occupied */
--color-seat-checked-in:       #1e88e5;  /* cloud_2 plane_ckin — NEW */
--color-seat-boarded:          #43a047;  /* cloud_2 plane_boarded — NEW */
--color-seat-blocked:          #757575;  /* cloud_2 plane_blocked */
--color-seat-exit-available:   #ffa726;  /* cloud_2 exit row — amber */
--color-seat-exit-occupied:    #e64a19;  /* cloud_2 occupied exit row */
--color-seat-unavailable:      #f5f5f5;  /* cloud_2 plane_m_s_ua */
--color-seat-infant:           #ce93d8;  /* cloud_2 plane_occupied_inf */

/* Seat cell geometry — cloud_2 plane_middle 30×33px, plane_side 50×40px */
--seat-size:       1.875rem;  /* 30px @ 16px root — was 2rem */
--seat-size-wide:  3.125rem;  /* 50px — reserved for future side-aisle cells */
--seat-radius:     4px;       /* unchanged */
```

Preserve any other tokens (typography, spacing, brand) unchanged. Keep ordering: colors first, geometry second, matches the file's existing pattern.

WHY: Token-level changes propagate everywhere via CSS custom properties — no per-component code change is required for the color shift. Adding `--color-seat-checked-in` and `--color-seat-boarded` lets Plan 08-01's CSS drop its `var(--token, fallback)` fallbacks (Task 2).

AVOID:
- Do not delete tokens that other parts of the app use (search-and-replace risk). Use Edit tool with precise old/new ranges, not a wholesale rewrite.
- Do not change `--seat-radius` — cloud_2 uses 4px and so do we.
- Do not introduce per-state radius tokens unless cloud_2 actually varies them (it does not).
- Do not touch `--color-bg`, `--color-border`, `--color-seat-text-*`, `--seat-gap`, `--font-*` — out of scope.
- Do not adjust legend swatch sizes here (CSS for swatches lives elsewhere).
  </action>
