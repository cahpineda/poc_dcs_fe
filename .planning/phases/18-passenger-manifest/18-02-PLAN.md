<objective>
Implement the **SEAT FILTERS** panel from the screenshot — 11 checkboxes that, when checked, dim non-matching available seats on the seat map. Phase 18-02 closes the second-largest gap from the screenshot analysis (after the passenger list, which 18-01 covers).

Architectural separation across three TDD-disciplined tasks:
1. **Domain**: a `SeatFilter` type and a deterministic match function — pure, fully unit-testable.
2. **Boundary**: extend the API contract minimally with a `seat_attributes` array per seat; map it through; populate the mock fixtures.
3. **Presentation**: render the panel, propagate active filters down the seat tree, dim non-matching cells.

The filter list is fixed (the 11 cloud2 filters from the screenshot). When zero filters are active, no seats are dimmed (default state). When ≥1 filter is active, available seats whose attributes do not include ANY of the active filter ids are dimmed (OR semantics — matches the screenshot's behavior of "show me ALL seats that satisfy any selected criterion"). If a stricter AND semantic is preferred, the user can request a Phase 18-03 follow-up — but OR is the documented default and matches industry seat-map conventions.
</objective>

    export const SEAT_FILTER_IDS = [
      'extra_leg_room', 'stretcher', 'no_smoke', 'infant_eligible',
      'exit_emergency', 'bulkhead', 'galley_proximity', 'usb_power',
      'window', 'window_no_view', 'other'
    ] as const;
    export type SeatFilterId = typeof SEAT_FILTER_IDS[number];

    export const SEAT_FILTER_LABELS: Record<SeatFilterId, string> = {
      extra_leg_room: 'Extra Leg Room',
      stretcher: 'Stretcher',
      no_smoke: 'No smoke seat',
      infant_eligible: 'Seat Available for adult with infant',
      exit_emergency: 'Exit and emergency exit',
      bulkhead: 'Bulkhead seat',
      galley_proximity: 'Galley Proximity',
      usb_power: 'USB Power Port',
      window: 'Window seat',
      window_no_view: 'Window seat without window',
      other: 'Other',
    };

    export function matchesFilters(seat: Seat, active: ReadonlySet<SeatFilterId>): boolean {
      if (active.size === 0) return true;
      if (!seat.isAvailable()) return true; // filters only affect available seats
      for (const id of active) {
        if (seat.matchesFilter(id)) return true;
      }
      return false;
    }
    ```

      matchesFilter(id: SeatFilterId): boolean {
        // Derived attributes (computed, not from fixture)
        if (id === 'exit_emergency' && this.isExitRow()) return true;
        if (id === 'window') {
          const num = this._number.toString();
          const col = num.charAt(num.length - 1);
          if (col === 'A' || col === 'F') return true;
        }
        return this._attributes.includes(id);
      }
      ```
