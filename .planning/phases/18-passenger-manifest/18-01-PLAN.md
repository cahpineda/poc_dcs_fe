<objective>
Implement the **left passenger sidebar** — the biggest remaining functional gap against cloud2. The panel shows every passenger on the flight as a list with name (LASTNAME,FIRSTNAME), check-in status badge, optional yellow highlight when their seat is currently active, a passenger count at the bottom, and click-to-jump-to-seat behavior.

Per the user constraint: no new API endpoints. The mock server already embeds passenger data per seat row in the existing seat plan response. Phase 18-01 derives the passenger list from the existing `SeatPlanResult` that `useSeatPlan` already returns — single source of truth, no duplicate state.

This plan is split into three TDD-disciplined tasks because each layer (domain → component → integration) has a distinct testable contract:
1. **Domain projection** — tests prove `derivePassengers` returns the correct set from any SeatPlanResult.
2. **Presentation** — tests prove PassengerList renders the projection, displays badges, and fires onSelect.
3. **Integration** — wires the list into SeatPlanTab and connects click → existing reseat flow.

Out of scope for 18-01 (lives in 18-02): SEAT FILTERS checkbox panel.
Out of scope entirely (deferred): real check-in workflow, "Check-In Group F2" / "Finish and Search Passengers F9" buttons, F-key keyboard shortcuts. Phase 18 only renders the manifest read-only and wires the click-to-seat behavior.
</objective>

    **Implementation** (`src/domain/seat/Passenger.ts`):
    ```ts
    export class Passenger {
      private constructor(
        readonly passengerKey: string | null,
        readonly name: string,
        readonly currentSeatNumber: string,
        readonly boardingGroup: number | null,
        readonly pnr: string | null,
        readonly gender: 'M' | 'F' | 'U' | null,
        readonly ssrs: string[],
        readonly hasInfant: boolean,
        readonly rushStatus: boolean,
        readonly isCheckedIn: boolean
      ) {}

      static fromSeat(seat: Seat): Passenger | null {
        if (!seat.passengerName) return null;
        const isCheckedIn = seat.status === 'checked_in' || seat.status === 'boarded';
        return new Passenger(
          seat.passengerKey,
          seat.passengerName,
          seat.number.toString(),
          seat.boardingGroup,
          seat.pnr,
          seat.gender,
          seat.ssrs,
          seat.hasInfant,
          seat.rushStatus,
          isCheckedIn
        );
      }

      get checkInLabel(): string { return this.isCheckedIn ? 'Checked' : 'Unchecked'; }

      get formattedName(): string {
        const words = this.name.trim().split(/\s+/);
        if (words.length < 2) return this.name;
        const last = words[words.length - 1];
        const rest = words.slice(0, -1).join(' ');
        return `${last.toUpperCase()},${rest.toUpperCase()}`;
      }

      equals(other: Passenger): boolean {
        if (this.passengerKey && other.passengerKey) return this.passengerKey === other.passengerKey;
        return this.name === other.name && this.currentSeatNumber === other.currentSeatNumber;
      }
    }
    ```

    **Add `derivePassengers()` to SeatPlanResult**:
    ```ts
    derivePassengers(): Passenger[] {
      const out: Passenger[] = [];
      for (const row of this.rows) {
        for (const seat of row.seats) {
          const p = Passenger.fromSeat(seat);
          if (p) out.push(p);
        }
      }
      return out;
    }
    ```
